🎯 Claude Code 只需按 HANDOFF.md 10 步执行

  1. 放入资产文件：cp handoff/xaml/*.xaml src/DesktopFences.UI/Themes/
  2. 重新生成 app.ico：ImageMagick 从 app-logo.svg 生成 9 尺寸 ICO，覆盖 Assets/app.ico
  3. App.xaml 合并字典：追加 AppLogo.xaml 与 Icons.xaml
  4. FencePanel.xaml：替换标题栏折叠、设置、新建 Tab、关闭按钮
  5. SettingsWindow.xaml：Logo + 三个窗口按钮 + 六个侧栏导航项
  6. SearchWindow.xaml：搜索框前缀 + 空态插画
  7. RuleEditorWindow.xaml：关闭按钮
  8. 所有 ContextMenu.MenuItem.Icon：10 种 emoji → Geometry
  9. NotifyIcon：从 DrawingImage 渲染托盘图标
  10. 构建验证：dotnet build + dotnet test（保持 61 个测试通过）
