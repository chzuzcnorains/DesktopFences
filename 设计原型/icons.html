三套并行的图标视觉语言，每一套都覆盖：应用主图标（多尺寸）、文件类型图标（14 种）、UI 操作图标（15 种）。整体继承项目现有的 Acrylic + oklch 248° 蓝色基调，可通过下方调色板预览不同主题色。

// 落地建议
· 方案 A「Fluent Acrylic」最贴合 Windows 11 Fluent Design，适合作为产品最终出厂方案
· 方案 B「Tile」最醒目、品牌识别度最强，可考虑用于任务栏/托盘与宣传物料

· 实际落地时可混合：主图标用 A/B 任一，内部功能图标统一用 C
